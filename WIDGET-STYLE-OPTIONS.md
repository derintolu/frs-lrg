# Widget Style Options for Market Matters & Announcements

## Current Issue
- Gradient text appears as solid blue color on dark backgrounds
- App launcher icons look dull
- Tiles need more visual pop

## Option 1: Bold Brand Accents
**Tiles:**
- Keep dark background but add **colored left border** (4px solid brand gradient)
- Make titles **solid white** (bold and clean)
- Add **colored icon backgrounds** (small circle behind icons with gradient)

**App Icons:**
- Add **gradient background** to each icon tile
- Increase **icon size** slightly
- Add **hover glow effect**

## Option 2: Subtle Color Tint
**Tiles:**
- Use **very subtle brand color tint** on backgrounds (blue/teal at 5-10% opacity)
- Keep **white text** for better readability
- Add **colored accent bar** at top of each tile

**App Icons:**
- Add **subtle gradient overlay** on hover
- Brighten existing colors
- Add **soft drop shadow**

## Option 3: Clean Minimal
**Tiles:**
- **Pure white text** - no gradients
- Increase **text size** for mortgage rates (larger, bolder)
- Add **colored underline** or **dot indicator** for accents
- Brighter **border colors** (white/40% instead of white/20%)

**App Icons:**
- Keep current colors
- Add **white border** around each icon
- Increase **contrast** on icon backgrounds

## Option 4: Icon-First Design
**Tiles:**
- Add **large colored icons** next to titles (trending arrows, bell, etc.)
- Make icons use **full gradient**
- Keep text simple and white
- Reduce visual noise

**App Icons:**
- **Larger icons** with less padding
- Add **subtle animation** on hover
- Brighten colors by 20%

## My Recommendation: Option 3 (Clean Minimal)

**Why:**
- Best readability with pure white text
- Clean, professional look
- Works well with dark theme
- No gradient text readability issues
- Easy to implement
